# Text Analytics Tool (Python)

A simple yet effective **command-line Text Analytics Tool** built using Python.  
This project analyzes raw text input to extract meaningful statistics such as word count, sentence count, common words, sentiment, and reading difficulty.

- Allows users to **save analysis results** to a text file
- Handles punctuation and case normalization manually
